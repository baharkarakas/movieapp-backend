How to Use

This page explains the purpose of the project and how to use the application.

Purpose

Movie Collection is a simple web application where users can discover movies, like them, and build a personal watchlist organized by categories.

How it works (Step-by-step)

  • Explore: Open the Home page to see all movies.
  • Register / Login: Create an account or sign in to unlock personal features.
  • Like movies: Click the heart button to like/unlike movies.
  • Create categories: Go to Dashboard and add categories (e.g., Sci-Fi, Drama).
  • Add to watchlist: Choose a category and click + Listeye Ekle.
  • View your lists: Dashboard shows your categories, watchlist items, and liked movies.
  • Logout: End your session anytime from the top menu.

Notes

  • You must be logged in to like movies or add them to your watchlist.
  • Password must be at least 6 characters.
